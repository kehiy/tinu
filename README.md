# tinu
simple URL shortener with Golang and fiber

## Tinu stands for => tiny URL

`
NOTE: this repo is still in development all of the stuff can change rapidly!
`
## usage:

1:
```bash
go install ./...
```

2:
```bash
go run main.go
```
### Docker:

1:
```bash
docker build -t tinu .
```

2:
```bash
docker run -p 3000:3000 tinu
```
